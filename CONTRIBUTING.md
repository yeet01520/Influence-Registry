# Contributing to The Influence Registry

Thank you for your interest in helping improve The Influence Registry. This project is about making public data more accessible, and contributions of all kinds are welcome.

## What we're looking for

The most valuable contributions are:

### Data corrections
If you spot something wrong, open an issue or PR with a public source backing the correction. Common examples:
- Wrong bioguide ID
- Outdated profile information after a member retires, resigns, or wins a special election
- Incorrect donation figures
- Broken or wrong photo for a member
- Missing controversies or conflicts of interest with public reporting attached

### New data sources
Suggestions for additional public, free datasets that would add value. Examples worth discussing:
- State-level lobbying data
- Foreign Agents Registration Act (FARA) filings
- Stock trading disclosures from new sources
- International equivalents
